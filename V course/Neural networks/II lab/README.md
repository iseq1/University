# Лабораторная работа №2

### Создание и обучение простой CNN-модели

Задача:
-  Взять [датасет данных](https://www.kaggle.com/datasets/thefamousrat/synthetic-chess-board-images/data);
-  На вход подаются пласт изображений шахматных досок с фигурами в разных положениях; 
-  Попробовать обучить простую CNN-модель для следующих задач:
   1. Определить количество фигур на шахматной доске
   2. Определить каких фигур больше: белых или черных
   3. У какой из сторон суммарная ценность фигур больше: у белых или у черных
- На выходе предоставить одно единственное значение


Реализация:
- Реализованы i и ii задачи
- Сформированы дополняющие результат графики и логи
