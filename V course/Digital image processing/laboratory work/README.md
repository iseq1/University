# Desktop-приложения для взаимодействия с медиа файлами

Это приложения для работы с изображениями формата __png/jpeg/bmp__. Преобразования, получение статистики, изменение исходного файла - все функциональности выполняются в рамках дисциплины **"Цифровая обработка изображений"**

---

## Функциональности

### 1. Приложение на основе меню выбора для взаимодействия с изображением:
- чтение и сохранение на диск файлы формата '.png', '.jpeg', '.bmp';'
- преобразование RGB-изображение в массив байтов и обратно;
- отображение текущих координат пикселя и значений R- G- B- компонент его амплитуды;
- Задание ROI (Регион интереса *(англ. Region of interest)*);
- отмена последнего выполненного действия на форме;

### 2. Замена цветного RBG-изображения на изображение в градациях серого с 24-разрядной кодировкой пикселя;

### 3. Вычисление минимальной и максимальной амплитуд, выборочных среднего и стандартного отклонения амплитуд пикселей, образующих как всю область изображения, так и только его границу;

### 4. Определение гистограммы амплитуд пикселей изображения/ROI или только границ. Отображение результата в отдельном окне;

### 5. Линейное контрастирование изображения/ROI. Отображение результата в отдельном окне;

### 6. Сглаживание амплитуд пикселей изображения/ROI. Отображение результата в отдельном окне;

### 7. Изменение масштаба изображения двумя методами: выборкой (метод ближайшего соседа) и билинейной интерполяции;

### 9.1 Получение и изменение амплитуды пикселя по заданным координатам.

### 9.2 Построение изображения с кусочно-постоянными амплитудами (карт).

### 10. Построение произвольного изображения с взаимно независимыми амплитудами (Распределение амплитуд: равномерного на интервале [a, b] и нормального со средним m и дисперсией σ^2).

### 11. Поворот изображения посредством аффинных преобразований.

### 12. Построение произвольного изображения методом скользящего суммирования по квадратной окрестности с радиусом r, используя в качестве исходника изображение с взаимно независимыми амплитудами и нормальным распределением N(m, σ^2 ) каждой амплитуды.

### 13. Оценка корреляционной функции по изображению прямоугольной области однородного случайного поля.
