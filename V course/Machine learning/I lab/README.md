# Лабораторная работа №1 — Классификация текстов и CBOW

## О проекте
Этот проект реализует две части лабораторной работы по обработке текстов и машинному обучению:

1. **Классификация текстов с использованием TF-IDF и логистической регрессии**  
   - Корпус: Steam отзывы на игры  
   - Модель: `LogisticRegression` (многоклассовая)  
   - Цель: классификация отзывов по 4 категориям (`positive_useful`, `positive_not_useful`, `negative_useful`, `negative_not_useful`)  
   - Используется TF-IDF для преобразования текста в векторы  

2. **Обучение word embedding с нуля методом CBOW**  
   - Корпус: тот же Steam  
   - Алгоритм CBOW реализован с нуля с помощью `numpy`  
   - Цель: построить эмбеддинги слов и проверить семантические аналогии (например, `good - bad + terrible ≈ fantastic`)
